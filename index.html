Consensus 1

Proof-of-work

POW

Cryptographic hash algorithm based problem solvers (Miners) compete to discover the hash of the next block.

The one with the higher computer power is more likely to find the hash quicker and get the reward.

Consensus 2

Proof-of-stake

POS

System arbitrary selects a node as a
validator for the next block.

The one with the biggest stake is more likely to be selected and get the reward.
